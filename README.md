# Code Keys (DEMO)

Code Keys is a browser-based interactive typing game.
This project showcases my ability to design and implement structured logic, manage user input and to combine both visual and audio feedback to create an interactive browser experience.
Each keypress triggers sound effects and visual feedback, with multiple levels, accuracy tracking and a results screen.

## Live Demo
https://consolejn.github.io/code-keys/
(NOTE: NOT ALL SOUNDS HAVE BEEN ADDED YET)

## Features
- Multi-level gameplay
- Sound effects per key
- Animated feedback
- Accuracy + timing system
